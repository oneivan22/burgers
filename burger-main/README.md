# Burger King Clone

Учебный проект — клон сайта Burger King Russia, разработанный на React.

## 📋 Технологии

- **React** — библиотека для создания UI
- **React Router** — для навигации между страницами
- **CSS Modules** — для стилизации компонентов
- **Context API** — для управления состоянием корзины
- **Vite** — сборщик проекта

## 🚀 Запуск проекта

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск dev-сервера

```bash
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

### 3. Сборка для продакшена

```bash
npm run build
```

### 4. Предпросмотр собранной версии

```bash
npm run preview
```

## 📁 Структура проекта

```
src/
├── components/         # UI-компоненты
│   ├── Header/        # Шапка сайта
│   ├── Footer/        # Подвал сайта
│   ├── ProductCard/   # Карточка товара
│   ├── PromoSlider/   # Промо-слайдер
│   ├── Loader/        # Индикатор загрузки
│   └── Error/         # Сообщение об ошибке
├── pages/             # Страницы
│   ├── HomePage/      # Главная страница
│   ├── MenuPage/      # Каталог товаров
│   ├── ProductPage/   # Страница товара
│   ├── CartPage/      # Корзина
│   └── LoginPage/     # Авторизация
├── context/           # Контексты (CartContext)
├── services/          # API-сервис (api.js)
├── data/              # Моковые данные (products.js)
├── assets/            # Изображения, шрифты
├── App.jsx            # Главный компонент с роутингом
└── main.jsx           # Точка входа
```

## 🌐 Страницы

### Главная (`/`)
- Stories с акциями
- Sidebar с категориями
- Популярные товары
- Переключение категорий

### Каталог (`/menu`)
- Все товары с фильтрацией по категориям
- Сортировка по цене (возрастание/убывание)
- Поиск по названию и описанию
- Индикатор загрузки и обработка ошибок

### Страница товара (`/product/:id`)
- Подробная информация о товаре
- Пищевая ценность (калории, белки, жиры, углеводы)
- Кнопка "Добавить в корзину"

### Корзина (`/cart`)
- Список добавленных товаров
- Изменение количества (+/−)
- Удаление товаров
- Итоговая сумма заказа
- Имитация оформления заказа (модальное окно)
- Сохранение в localStorage

### Авторизация (`/login`)
- Форма с логином и паролем
- Валидация полей
- Сохранение состояния в localStorage
- Имитация задержки при входе

## 📱 Адаптивность

- **Desktop**: ≥1200px — полная версия
- **Tablet**: 768–1199px — 2 колонки товаров
- **Mobile**: ≤767px — 1-2 колонки, горизонтальный скролл категорий

## 📦 Данные

Моковые данные о товарах находятся в `src/data/products.js`. Включают:
- 20 товаров (бургеры, курица, рыба, закуски, напитки, комбо, десерты, соусы)
- 8 категорий
- 3 промо-баннера

API-запросы имитируются через `src/services/api.js` с задержкой для демонстрации loading/error состояний.
